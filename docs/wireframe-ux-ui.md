# Wireframe / UX / UI 설계

## Wireframe (텍스트)
- Header: 로고 + 글로벌 네비게이션
- Hero: 비전/핵심 가치
- Content Grid: 13개 메뉴 카드
- Detail Page: 설명 + 2열 카드(핵심 항목/체크리스트)
- Admin: 좌측 메뉴 + 우측 폼

## UX 설계
- 탐색 우선: 메뉴를 명확한 도메인 단위로 고정
- 검색 우선: 법규/판례/사례 통합 검색 엔드포인트
- 신뢰 우선: 출처/개정이력/기관 정보 노출
- 모바일 우선: 단일 컬럼 카드 스택

## UI 디자인 원칙
- 네이비 + 화이트 기반의 미니멀 톤
- 보조색은 슬레이트 계열로 제한
- 타이포는 가독성 중심, 정보 위계 명확화
- 다크모드 대응
