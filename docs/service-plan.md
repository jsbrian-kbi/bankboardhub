# Bank Board Governance Hub 서비스 기획서

## 1) 비전
- 국내 최고의 은행/은행지주 이사회 및 사외이사 전문 지식 플랫폼
- 데이터 기반 + 실무 중심 + 거버넌스 전문 + AI 지식 지원

## 2) 타깃 사용자
- 은행·은행지주 사외이사, 예비 사외이사, 감사위원, 위험관리위원
- 이사회 사무국, 지배구조 담당자, 연구자

## 3) 제품 원칙
- 독립 서비스(특정 금융회사 홈페이지 비연계)
- 규제·사례·판례·교육·뉴스의 단일 정보 허브
- 출처 표시와 개정이력 중심의 신뢰성 설계

## 4) 핵심 기능
- 메뉴형 지식 허브(13개 메뉴)
- 법규/사례/판례 Full Text Search
- 관리자 입력 워크플로우(등록/수정/권한)
- AI Board Assistant(질의응답 + 출처)

## 5) 단계별 로드맵
- Phase 1: 정보구조, UI 시스템, 데이터 모델, 관리자 CRUD
- Phase 2: 검색 고도화, RAG, 타임라인/대시보드
- Phase 3: 개인화 추천, 위원회별 리스크 알림
